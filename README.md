# Pepe Docker

# Install

```console
# git clone --recursive https://github.com/pepeproject/pepe-docker.git
# cd pepe-docker
## MACOS: brew install git-lfs
## LINUX: See https://github.com/git-lfs/git-lfs/releases
# git lfs install
# git lfs pull
```

## Using

```console
# make dist
# make run

...

# make stop
# make volume-prune

```
